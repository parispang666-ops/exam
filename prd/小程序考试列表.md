# 小程序考试列表 PRD（产品需求文档）

## 1. 文档范围

本文档描述小程序端（C 端考生视角）"考试列表"页面的完整功能，包括本租户内考试展示（含上级派发考试）、考试状态切换、搜索筛选、结果状态回显和人员调动场景下的考试可见性规则。

小程序端仅面向考生本人，不涉及管理员操作（创建、编辑、删除、发布等）。

> 本页面 PRD 需配合[《考试功能-整体PRD》](./考试功能-整体PRD.md)阅读，全局业务规则、数据模型、角色权限定义和核心流程参见整体 PRD。

## 2. 使用角色

- **考生（员工）**：在小程序端查看本人当前租户下的考试列表，包含进行中、未开始、已结束三种状态的考试，可查看考试结果（及格/不及格/未参加）和得分。
- **系统**：根据考生所属租户类型（集团/下级单位/独立租户）和考试来源（自建/上级下发），筛选当前租户下该考生应展示的考试。

## 4. 功能清单

| 模块 | 功能 | 功能描述 |
|------|------|----------|
| 考试列表 | 多状态 Tab 切换 | 支持「进行中」「未开始」「已结束」三个状态 Tab 切换查看。 |
| 考试列表 | 混合考试来源展示 | 下级单位考生同时展示本单位自建考试和上级（集团）派发考试。 |
| 考试列表 | 搜索筛选 | 支持关键词搜索考试名称；支持筛选条件过滤。 |
| 考试卡片 | 考试概要信息 | 展示考试结束时间、考试时长、缩略图、分类角标、考试名称、副标题。 |
| 考试卡片 | 结果状态回显 | 展示考生本人完成状态：及格（绿色）/ 不及格（红色）/ 未参加（橙色），及具体得分。 |
| 考试可见性 | 人员调动规则 | 考生调出当前租户后，原租户定向考试自动隐藏；调回后恢复展示。详见第 5 节。 |

## 5. 需求说明

### 5.1 考试来源与展示规则

各租户类型下小程序端考试展示范围：

| 租户类型 | 展示的考试来源 | 说明 |
|----------|---------------|------|
| 集团 | 本单位考试 | 展示集团本部为该考生下发的考试（含定向指定人员和部门范围覆盖）。 |
| 下级单位 | 本单位自建考试 + 上级（集团）下发考试 | 同时展示本级自建考试和集团派发至本单位的考试。上级下发考试与自建考试在列表中混合展示，不做视觉区分。 |
| 独立租户 | 本单位考试 | 仅展示本租户自建考试。 |

> 小程序端面向考生本人，列表中的考试均为该考生当前租户下应参加的考试。上级派发考试在下级单位考生列表中与自建考试无视觉区分，统一展示。

### 5.2 状态 Tab 切换

| Tab | 筛选逻辑 | 说明 |
|-----|---------|------|
| 进行中 | 当前时间在考试开始时间与结束时间之间 | 默认选中 Tab。展示正在进行的考试。 |
| 未开始 | 当前时间早于考试开始时间 | 展示尚未开始的考试，考生可提前查看但不可作答。 |
| 已结束 | 当前时间晚于考试结束时间 | 展示已过期的考试，考生可查看结果。 |

- 默认进入页面时选中「进行中」Tab。
- 切换 Tab 时列表数据重新加载。
- 空数据时展示空状态提示。

### 5.3 搜索与筛选

#### 5.3.1 搜索

- 搜索框支持输入关键词，按考试名称模糊匹配。
- 搜索范围为当前 Tab 状态下的考试列表。
- 清除搜索关键词后恢复完整列表。

#### 5.3.2 筛选

- 点击筛选按钮展开筛选条件面板。
- 筛选条件可包含：考试来源（自建/上级下发，仅下级单位可选）。
- 筛选与搜索关键词叠加生效。

### 5.4 考试卡片

每张考试卡片展示以下信息：

| 区域 | 字段 | 说明 |
|------|------|------|
| 顶部信息栏 | 考试结束时间 | 格式：`YYYY-MM-DD HH:mm 结束`。已结束考试展示实际结束时间。 |
| 顶部信息栏 | 考试时长 | 以时钟图标 + 分钟数展示，如「90 分钟」。 |
| 左侧缩略图 | 课程封面图 | 考试关联课程的缩略图，含分类角标（如「专项题」「2025」「精品课」）。 |
| 左侧缩略图 | 课程标题 | 缩略图上叠加的课程标题文字。 |
| 右侧信息区 | 考试名称 | 考试主标题，单行截断，超长显示省略号。 |
| 右侧信息区 | 考试副标题 | 考试分类或来源说明，如「安全生产考试」。 |
| 右侧信息区 | 结果标签 | 已完成的考试展示结果：**及格**（绿色 + 得分）/ **不及格**（红色 + 得分）/ **未参加**（橙色，无得分）。 |

#### 5.4.1 结果状态

| 状态 | 标签颜色 | 得分展示 | 触发条件 |
|------|:---:|------|----------|
| 及格 | 绿色 `#19C6B7` | 展示具体得分（如「得分 100/100」） | 考生已完成考试且分数 ≥ 及格线 |
| 不及格 | 红色 `#FF5B4D` | 展示具体得分（如「得分 59/100」） | 考生已完成考试且分数 < 及格线 |
| 未参加 | 橙色 `#FF7A3D` | 不展示得分 | 考试已结束或进行中，考生尚未作答 |

### 5.5 人员调动场景下的考试可见性

参考规则说明第 3 节「用户考试活动展示规则」：

| 场景 | 规则 |
|------|------|
| 正常状态 | 仅当考生当前所属租户 = 考试下发时该考生归属的租户，才在列表中展示。 |
| A 调动至 B | 原 A 公司定向给该考生、且未作答未过期的考试直接隐藏，不再展示。仅展示当前 B 租户下定向给该考生的考试任务。 |
| A → B → A（调回） | 考生从 B 调回原 A 公司后，原先在 A 的定向考试重新在列表中恢复展示。 |
| 个人档案 | 不受租户调动影响，永久留存全部本人参与过的考试记录（含调动前完成的考试），统一在「个人档案」栏目查看。 |

### 5.6 空状态

- 当前 Tab 下无考试数据时展示空状态提示文案，如「暂无进行中的考试」。
- 搜索无结果时展示「未找到相关考试」。

## 6. 交互设计

### 6.1 页面加载

1. 进入页面，默认选中「进行中」Tab。
2. 请求当前租户下该考生的进行中考试列表。
3. 列表按考试结束时间正序排列（即将结束的在前）。

### 6.2 Tab 切换

1. 考生点击「未开始」或「已结束」Tab。
2. Tab 切换为选中态（蓝色文字 + 底部蓝色指示条）。
3. 列表数据按对应状态重新加载。
4. 搜索关键词和筛选条件在切换 Tab 后保留（如适用）。

### 6.3 搜索交互

1. 考生在搜索框输入关键词。
2. 实时筛选 / 或点击搜索按钮后过滤列表。
3. 搜索结果在当前 Tab 状态下展示。

### 6.4 筛选交互

1. 点击右侧筛选按钮，展开筛选面板。
2. 选择筛选条件后，列表按条件过滤。
3. 筛选条件可叠加搜索关键词。

### 6.5 考试卡片点击

- 点击进行中/未开始的考试卡片 → 进入考试详情页或直接开始作答（具体跳转取决于考试配置）。
- 点击已结束的考试卡片 → 查看考试成绩详情。

## 7. 数据依赖

### 7.1 考试列表接口

需返回字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| 考试ID | string | 唯一标识 |
| 考试名称 | string | 考试主标题 |
| 考试副标题 | string | 如「安全生产考试」 |
| 考试来源 | string | 自建 / 上级下发 |
| 考试状态 | string | 未开始 / 进行中 / 已结束 |
| 考试开始时间 | datetime | |
| 考试结束时间 | datetime | |
| 考试时长 | int | 单位：分钟 |
| 缩略图URL | string | 课程封面图 |
| 缩略图分类角标 | string | 如「专项题」「2025」「精品课」 |
| 缩略图课程标题 | string | 叠在缩略图上的课程名称 |

### 7.2 考生结果接口

需返回字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| 考试ID | string | 关联考试 |
| 考生ID | string | 当前用户 |
| 完成状态 | string | 已完成 / 未完成 |
| 分数 | int | 得分 |
| 是否及格 | bool | |
| 交卷时间 | datetime | |

## 8. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 网络请求失败 | 展示加载失败提示，提供「重试」按钮。 |
| 考试列表为空 | 展示空状态提示文案。 |
| 搜索无结果 | 展示「未找到相关考试」提示。 |
| 考试已过期但无结果数据 | 按「未参加」状态展示，不展示得分。 |

## 9. 非功能性需求

| 指标 | 要求 |
|------|------|
| 列表加载响应 | ≤ 2s |
| 搜索响应 | ≤ 500ms（本地过滤） |
| Tab 切换响应 | ≤ 1s |
| 适配屏幕 | 375pt 及以上宽度（小程序标准） |

## 10. 验收标准

| 验收项 | 验收标准 |
|--------|----------|
| Tab 切换 | 「进行中」「未开始」「已结束」三个 Tab 切换正常，数据按状态正确筛选。 |
| 默认 Tab | 进入页面默认选中「进行中」。 |
| 考试来源（下级单位） | 下级单位考生列表同时展示本单位自建考试和上级派发考试。 |
| 考试来源（集团/独立租户） | 集团和独立租户考生列表仅展示本单位考试。 |
| 搜索 | 输入关键词可正确筛选考试名称。 |
| 筛选 | 筛选条件正确过滤列表。 |
| 结果状态-及格 | 已完成且及格的考试展示绿色「及格」标签 + 得分。 |
| 结果状态-不及格 | 已完成且不及格的考试展示红色「不及格」标签 + 得分。 |
| 结果状态-未参加 | 未作答的考试展示橙色「未参加」标签。 |
| 考试卡片信息 | 卡片正确展示结束时间、时长、缩略图、角标、考试名称、副标题。 |
| 空状态 | 无数据时展示对应空状态提示。 |
| 人员调动 | 考生调出后原租户定向考试隐藏，调回后恢复展示。 |
