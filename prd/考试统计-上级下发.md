# 考试统计-上级下发 PRD

## 1. 文档范围

本文档描述"考试统计-上级下发"页面的完整需求。该页面用于下级单位管理员查看**上级（集团）派发考试**在本单位的统计数据，涵盖**统计范围、手机号字段、人员状态、部门统计人员过滤**等核心需求。普通统计卡片（及格率/参与率/分数概览）、考试记录列表、导出、分页等基础统计能力沿用既有考试统计规则，本文不重复说明。

页面原型文件：`考试统计-下级单位:独立租户.html`

> 本页面PRD需配合[《考试功能-整体PRD》](./考试功能-整体PRD.md)阅读，全局业务规则、数据模型、角色权限定义和核心流程参见整体PRD。

## 2. 业务背景

在集团-下级单位体系下，集团管理员可向多个下级单位派发考试。下级单位管理员在考试列表的"上级下发"分类中可看到这些考试，并通过"管理"按钮进入本统计页面，查看该上级下发考试在本单位及下属部门范围内的考试数据。

### 2.1 与"考试统计-本单位"的核心差异

| 差异维度 | 考试统计-本单位 | 考试统计-上级下发 |
|----------|:---:|:---:|
| 考试来源 | 本单位自建考试 | 上级（集团）下发考试 |
| 列表入口 | 全部分类 / 具体分类节点 | 上级下发分类节点 |
| 统计范围 | 本单位 + 下属部门 | 本单位 + 下属部门（上级考试可能覆盖多家单位，但下级单位仅看自己范围） |
| 考试信息 | 本单位管理员设定 | 上级管理员设定，仅可查看不可修改 |
| 返回导航 | 返回考试列表 | 返回考试列表 |

## 3. 使用角色

- **下级单位管理员**：在本单位及下属部门范围内查看上级下发考试的考试成绩、考试记录、部门统计，支持按人员状态筛选，部门统计默认仅统计在职人员。

## 4. 前置条件

- 用户已登录下级单位管理端
- 当前考试为上级（集团）下发的考试，考试已发布并生成完整的人员快照和考试记录
- 从考试列表点击上级下发考试的"统计"入口进入统计页面
- 该考试在下级单位的人员快照中包含本单位及下属部门的人员

## 5. 统计范围说明

上级下发考试由集团管理员创建，考生范围可能涵盖多家下级单位。**下级单位管理员查看统计时，系统仅展示本单位和下属部门的人员数据**，不展示其他单位的人员信息。

## 7. 功能清单

| 模块 | 功能 | 功能描述 |
|------|------|----------|
| 统计范围 | 仅限本单位和下属部门 | 上级下发考试虽覆盖多家单位，但下级单位仅展示本单位和下属部门的人员数据。 |
| 统计范围 | 全量人员保留 | 本单位考试快照中的全部人员均保留在统计列表中，离职、停用后人员记录不隐藏、不移除。 |
| 考试成绩 | 手机号 + 人员状态列新增 | 列表新增"手机号"列和"人员状态"列，展示在职/离职/停用，查询条件新增"人员状态"筛选。 |
| 考试记录 | 人员状态列新增 | 列表新增"人员状态"列，展示在职/离职/停用，查询条件新增"人员状态"筛选。 |
| 部门统计 | 只统计在职人员 | 筛选区新增"只统计在职人员"勾选项，默认勾选；取消勾选后纳入离职、停用人员。 |
| 导出 | 导出全部 / 批量导出 | 三个页签均提供导出下拉菜单：导出全部（导出当前列表全量数据）、批量导出（需勾选至少一条复选框数据，否则提示"请选择要导出的内容"）。导出文件为 Excel（.xlsx）格式，文件名规则：「考试名称-页签名称.xlsx」。列表首列提供全选/单选复选框。 |

## 8. 需求说明

### 8.1 统计范围

#### 8.1.1 单位数据隔离

| 规则项 | 页面要求 |
|--------|----------|
| 数据范围 | 仅展示当前下级单位及下属部门的人员数据。上级下发考试可能覆盖多家单位，但当前页面仅展示本单位的统计结果。 |
| 隔离机制 | 系统按当前登录单位的租户ID过滤人员快照和考试记录，确保不同下级单位之间数据不可互看。 |
| 概览卡片 | 及格率、参与率、分数概览等统计卡片，仅基于本单位及下属部门的人员数据计算。 |

#### 8.1.2 全量人员保留

| 规则项 | 页面要求 |
|--------|----------|
| 人员基准 | 统计列表基于上级下发考试发布时生成本单位的人员快照，包含该考试在本单位范围内的全部应考人员。 |
| 离职人员 | 考试发布后离职的人员，其姓名仍然保留在考试成绩和考试记录列表中，通过"人员状态"列标注为"离职"。 |
| 停用人员 | 考试发布后被停用的人员，其姓名仍然保留在考试成绩和考试记录列表中，通过"人员状态"列标注为"停用"。 |
| 不隐藏规则 | 离职、停用人员记录不被系统自动移除或隐藏，用户可通过查询条件和列表直接查看。 |
| 统计口径 | 离职、停用人员在考试成绩、考试记录中正常展示；在部门统计中受"只统计在职人员"勾选项控制。 |

### 8.2 考试成绩 / 考试记录：新增手机号与人员状态

#### 8.2.1 查询条件新增

| 规则项 | 规则 |
|--------|------|
| 位置 | 查询条件区域，位于"部门"筛选之后。 |
| 标签 | 统一使用"人员状态"。 |
| 下拉选项 | 请选择（默认）、在职、离职、停用。 |
| 默认值 | 请选择，即不按人员状态筛选，展示全部人员。 |
| 查询逻辑 | 选择具体状态值后点击"查询"，过滤出对应人员状态的记录。 |

#### 8.2.2 列表新增列

| 页签 | 列顺序 |
|------|--------|
| 考试成绩 | 姓名 → 部门 → 手机号 → 人员状态 |
| 考试记录 | 姓名 → 部门 → 人员状态 |

手机号列规则：

| 规则项 | 规则 |
|--------|------|
| 列名 | 手机号 |
| 适用页签 | 考试成绩。 |
| 位置 | 位于"部门"列之后、"人员状态"列之前。 |
| 展示内容 | 展示考生手机号。 |
| 缺失处理 | 手机号为空时显示"--"。 |

人员状态列规则：

| 规则项 | 规则 |
|--------|------|
| 列名 | 人员状态 |
| 位置 | 位于"部门"列之后、"完成状态/当次作答次数"列之前。 |
| 展示风格 | 使用彩色标签展示：在职—绿色标签、离职—红色标签、停用—橘黄色标签。 |
| 数据来源 | 取自人员快照中记录的人员状态或实时人员主数据状态（待确认具体取数逻辑）。 |

#### 8.2.3 人员状态枚举与样式

| 状态值 | 标签样式 | 说明 |
|--------|----------|------|
| 在职 | 绿色背景 + 绿色文字（`person-state`） | 人员当前在职/启用。 |
| 离职 | 红色背景 + 红色文字（`person-state left`） | 人员已离职，姓名保留在列表中。 |
| 停用 | 橘黄色背景 + 橘黄色文字（`person-state disabled`） | 人员已停用/禁用，姓名保留在列表中。 |

### 8.4 导出功能

#### 8.4.1 导出按钮与菜单

| 规则项 | 规则 |
|--------|------|
| 组件类型 | 导出下拉菜单按钮，三个页签（考试成绩、考试记录、部门统计）各有一个独立的导出按钮区域。 |
| 按钮文案 | "导出⌄"（带下拉箭头）。 |
| 菜单项 | 展开后显示两个选项：**导出全部**、**批量导出**。 |
| 菜单交互 | 点击导出按钮展开菜单；再次点击或点击页面空白处收起。同一时间仅一个菜单展开。 |

#### 8.4.2 复选框行选择

| 规则项 | 规则 |
|--------|------|
| 位置 | 每个页签的列表表格首列（序号列之前）为复选框列。 |
| 表头复选框 | 全选/取消全选：点击表头复选框，切换当前列表（tbody）全部行的选中状态。部分选中时自动同步状态。 |
| 行复选框 | 每行左侧有独立复选框，点击切换该行选中状态。选中行带 `.checked` 样式。 |
| 状态联动 | 当 tbody 内所有行复选框均选中时，表头复选框自动变为选中态；否则取消选中态。 |

#### 8.4.3 导出全部

| 规则项 | 规则 |
|--------|------|
| 触发 | 点击导出菜单中的"导出全部"。 |
| 行为 | 将当前页签列表中**全部数据行**（忽略复选框勾选状态）导出为 Excel（.xlsx）文件。 |
| 菜单关闭 | 点击后自动关闭导出菜单。 |
| 成功提示 | 导出完成后 toast 提示"导出成功"。 |

#### 8.4.4 批量导出

| 规则项 | 规则 |
|--------|------|
| 触发 | 点击导出菜单中的"批量导出"。 |
| 前置校验 | 检查当前页签 tbody 内是否有至少一个复选框被勾选。 |
| 未选中处理 | 若无任何行被选中，toast 提示"**请选择要导出的内容**"，不执行导出操作。 |
| 选中后导出 | 仅导出被勾选的数据行。导出完成后 toast 提示"导出成功"。 |
| 菜单关闭 | 点击后自动关闭导出菜单（无论是否触发导出）。 |

#### 8.4.5 导出文件规范

| 规则项 | 规则 |
|--------|------|
| 格式 | Excel 标准 .xlsx 格式，使用 SheetJS 生成，可直接用 Excel/WPS 打开，原生支持中文。 |
| 文件名 | `考试名称-页签名称.xlsx`，例：`集团消防安全专项考试-考试成绩.xlsx`、`集团消防安全专项考试-考试记录.xlsx`、`集团消防安全专项考试-部门统计.xlsx`。 |
| 页签名称映射 | 考试成绩页签 → `考试成绩`；考试记录页签 → `考试记录`；部门统计页签 → `部门统计`。 |
| 列处理 | 表头列和每行数据列均跳过首列（复选框列），从第二列开始导出。 |
| 特殊字符 | 单元格内容中的双引号转义为 `""`，每个字段值用双引号包裹。 |

### 8.3 部门统计：只统计在职人员

| 规则项 | 规则 |
|--------|------|
| 位置 | 部门统计页签的查询条件区域，位于"部门"筛选之后、"查询/重置"按钮之前。 |
| 组件类型 | 复选框 + 文字标签。 |
| 标签文案 | "只统计在职人员"。 |
| 默认状态 | **勾选**（`checked`），即默认只统计在职/启用人员。 |
| 勾选时 | 部门统计的各指标（应考人数、实考人数、缺考人数、及格人数、参与率、及格率、最高分、最低分、平均分）仅统计本单位范围内人员状态为"在职"的记录。 |
| 取消勾选时 | 统计范围扩大到本单位全部人员，包含在职、离职、停用人员，各指标基于全量数据重新计算。 |
| 切换行为 | 勾选/取消勾选后，需点击"查询"按钮触发数据刷新，而非即时生效。 |
| 影响范围 | 仅影响部门统计页签，不影响考试成绩和考试记录页签的数据展示。 |

## 9. 交互设计

### 9.1 页面入口

- 从下级单位考试列表页，在左侧分类树中点击"上级下发"节点，筛选出上级下发考试。
- 在上级下发考试所在行的操作列，点击"统计"按钮进入本统计页面。
- 上级下发考试的操作列仅展示"统计"按钮，不支持预览、编辑、删除、发布等操作。

### 9.2 人员状态筛选

- 在考试成绩或考试记录页签，选择"人员状态"下拉值并点击"查询"后，列表仅展示对应状态的记录。
- 选择"请选择"并查询，恢复展示全部人员。
- 点击"重置"，人员状态下拉恢复为"请选择"。

### 9.3 只统计在职人员切换

- 进入部门统计页签时，勾选项默认勾选，列表按在职口径展示。
- 用户取消勾选后点击"查询"，列表刷新为全量口径。
- 重新勾选后点击"查询"，列表恢复为在职口径。

### 9.4 页签切换

- 在考试成绩、考试记录、部门统计三个页签之间切换时，各页签的查询条件和数据状态保持独立。
- 考试成绩和考试记录的"人员状态"筛选互不影响。
- 部门统计的"只统计在职人员"勾选状态仅在部门统计页签内有效。

### 9.5 导出交互

- 点击"导出⌄"按钮展开下拉菜单，显示"导出全部"和"批量导出"两个选项。
- 点击"导出全部"：直接导出当前页签全部数据，忽略复选框状态。
- 点击"批量导出"：先校验是否有勾选行，无勾选则 toast 提示"请选择要导出的内容"；有勾选则仅导出选中行。
- 列表首列复选框支持全选（点击表头）和单选（点击行首），全选与单选状态联动。
- 导出文件名格式：「考试名称-页签名称.xlsx」，文件为 Excel（.xlsx）格式。
- 导出完成后 toast 提示"导出成功"。点击菜单外部区域或再次点击按钮关闭菜单。

### 9.5 返回导航

- 点击页面标题栏的"← 统计分析"返回按钮，返回上级下发考试列表页（保留分类树"上级下发"筛选状态）。

## 10. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 上级下发考试无本单位人员数据 | 概览卡片指标显示"0"或"0%"，列表展示空状态提示"暂无数据"。 |
| 人员快照数据缺失人员状态 | 人员状态列显示"--"，不作为筛选条件参与过滤。 |
| 部门统计全量口径下某部门无数据 | 该部门行正常展示，指标值显示"0"或"0%"。 |
| 切换页签时查询结果为空 | 列表展示空状态提示"暂无数据"。 |
| 当前用户无权限查看该考试统计 | 提示"无权限访问"，返回考试列表页。 |

## 11. 验收标准

| 验收项 | 验收标准 |
|--------|----------|
| 数据隔离 | 仅展示本单位和下属部门的人员数据，不可见其他单位数据。 |
| 离职人员保留 | 考试发布后离职的人员仍在考试成绩和考试记录列表中展示，人员状态显示"离职"。 |
| 停用人员保留 | 考试发布后停用的人员仍在考试成绩和考试记录列表中展示，人员状态显示"停用"。 |
| 考试成绩手机号列 | 考试成绩列表包含"手机号"列，位于部门之后、人员状态之前。 |
| 考试成绩人员状态列 | 列表包含"人员状态"列，展示在职/离职/停用，样式与枚举一致。 |
| 考试成绩人员状态筛选 | 查询条件包含"人员状态"下拉，选择后查询正确过滤。 |
| 考试记录人员状态列 | 列表包含"人员状态"列，展示在职/离职/停用，样式与枚举一致。 |
| 考试记录人员状态筛选 | 查询条件包含"人员状态"下拉，选择后查询正确过滤。 |
| 部门统计默认勾选 | 进入部门统计页签时，"只统计在职人员"默认勾选。 |
| 部门统计勾选口径 | 勾选时仅统计在职人员，各指标基于在职人员计算。 |
| 部门统计取消勾选 | 取消勾选后点击查询，统计包含全部人员（在职、离职、停用），各指标基于全量数据重新计算。 |
| 页签独立性 | 各页签的筛选条件和数据状态互不影响。 |
| 入口正确 | 从下级单位考试列表"上级下发"分类中点击"统计"进入本页面。 |
| 返回导航 | 点击返回按钮可返回上级下发考试列表页。 |
| 复选框全选 | 点击表头复选框可全选/取消全选当前页签所有数据行，选中行 `.ck` 添加 `.checked` 样式。 |
| 复选框状态联动 | tbody 内所有行均选中时表头复选框自动勾选，部分选中时表头复选框取消勾选。 |
| 导出全部 | 点击"导出全部"，生成当前页签全量数据的 Excel（.xlsx）文件，文件名「考试名称-页签名称.xlsx」。 |
| 批量导出（有选中） | 勾选至少一行后点击"批量导出"，仅导出选中行数据。 |
| 批量导出（无选中） | 未勾选任何行时点击"批量导出"，toast 提示"请选择要导出的内容"，不执行导出。 |
| 导出文件格式 | 生成标准 .xlsx 格式文件，兼容 Excel/WPS，原生支持中文无需额外编码处理。 |
