# 批量导入考生 PRD

## 1. 文档范围

本文档描述考试考生管理中的"批量导入考生"功能，适用于新增考试和编辑考试页面中需要批量添加考生到当前考试的场景。

适用页面：
- `新增考试-集团.html`
- `新增考试-下级单位.html`
- `新增考试-普通租户.html`
- `编辑考试-集团.html`
- `编辑考试-下级单位:独立租户.html`

## 2. 功能入口

| 项目 | 说明 |
|------|------|
| **按钮名称** | 批量导入 |
| **按钮位置** | "选择考生"按钮（页面显示为"选择"）旁边 |
| **按钮样式** | 次级按钮（白底、蓝边、蓝色文字，hover 变蓝底白字） |
| **可见条件** | 仅在"仅部分人员"模式下显示；"组织内全部人员"模式下隐藏 |

## 3. 数据流向

```
批量导入
↓
下载标准模板
↓
按模板填写考生信息
↓
选择 Excel 文件上传
↓
系统校验 + 去重
↓
导入成功 → 数据自动合并到"已选考生"列表
```

导入成功后，考生数据**自动添加**到"已选考生"列表中，无需手动刷新或二次确认。

## 4. 弹窗交互

弹窗标题为"导入"，包含以下两个步骤：

### 4.1 步骤一：下载模板，填写信息

- 展示 Excel 图标、说明文案"按照下载的模板，完成信息填写"
- 提供"下载模板"按钮，点击后下载 `.csv` 格式标准模板

### 4.2 步骤二：上传填好文件

- 展示上传区域，支持点击上传和拖拽上传
- 文案："将文件拖到此处，或 点击上传"；"最大支持5M、仅xls、xlsx的文件"
- 已选择文件后展示文件名和移除按钮
- 底部按钮："返回"（关闭弹窗）、"导入"（未选文件时禁用）

## 5. 模板说明

### 5.1 模板字段

| 字段 | 必填 | 字段说明 |
|------|------|----------|
| 姓名 | ✅ 必填 | 考生姓名，最大长度 10 个字符 |
| 手机号 | ✅ 必填 | 中国大陆手机号，11 位数字，以 1 开头 |

### 5.2 模板格式要求

| 项目 | 要求 |
|------|------|
| **文件格式** | .xlsx 或 .xls（原型阶段以 `.csv` 模板替代） |
| **表头行** | 第一行为表头：姓名、手机号 |
| **数据行** | 从第二行开始，每行一条考生数据 |
| **空行处理** | 忽略空行，不影响导入 |

## 6. 校验规则

### 6.1 文件级校验

| 错误编码 | 错误信息 | 触发条件 |
|----------|----------|----------|
| E-001 | 文件格式不支持，请上传 .xlsx 或 .xls 文件 | 上传的文件不是 Excel 格式 |
| E-002 | 模板格式不正确，请使用系统提供的标准模板 | 表头与模板不一致 |
| E-003 | 文件为空，请下载模板后填写数据 | Excel 内容为空 |
| E-004 | 导入数据超过最大限制（1000行），请分批导入 | 单次导入超过 1000 行 |
| E-005 | 文件损坏或无法解析，请检查文件后重试 | 文件损坏或无法读取 |

### 6.2 行级校验

| 错误编码 | 错误信息 | 触发条件 |
|----------|----------|----------|
| E-101 | 姓名为空，请填写姓名 | "姓名"列未填写内容 |
| E-102 | 姓名长度超过限制（最大10个字符），请调整 | 姓名超过 10 个字符 |
| E-103 | 手机号为空，请填写手机号 | "手机号"列未填写内容 |
| E-104 | 手机号格式错误，请填写11位数字，以1开头 | 手机号不符合规则 |
| E-105 | 数据行无任何内容，请删除空行后重试 | 整行数据为空 |

## 7. 去重规则

导入时自动去重，不因重复数据阻塞导入流程。

| 重复类型 | 处理方式 | 提示信息 |
|----------|----------|----------|
| **与已选考生重复** | 跳过该记录 | 已跳过（该考生已存在于已选考生列表中） |
| **Excel 内重复** | 保留第一条，后续跳过 | 已跳过（与本次导入数据重复） |

同一批次导入中，手机号相同的只保留最先出现的行，后续重复行自动跳过。

## 8. 导入结果

### 8.1 结果分类

| 分类 | 说明 | 是否展示在错误文件 |
|------|------|-------------------|
| **成功** | 校验通过，成功添加到已选考生列表 | ❌ 不展示 |
| **跳过（已存在）** | 数据重复，系统自动过滤，无需用户操作 | ❌ 不展示 |
| **失败** | 数据格式错误，需修正后重新导入 | ✅ 展示 |

### 8.2 结果提示

| 场景 | 弹窗/提示内容 | 操作 |
|------|--------------|------|
| 全部成功 | "成功导入 X 名考生" | 点击"确定"后关闭弹窗 |
| 部分成功（含跳过） | "成功导入 X 人，跳过 Y 人（已存在），失败 0 人" | 点击"确定"后关闭弹窗 |
| 有失败记录 | "成功导入 X 人，跳过 Y 人（已存在），失败 Z 人。是否下载错误文件查看原因？" | "确定"下载错误文件，"取消"留在弹窗 |
| 全部失败 | "导入失败，请下载错误文件查看原因并修正后重新导入" | 自动触发错误文件下载 |

### 8.3 错误文件

- 格式：`.csv`（原型阶段），正式环境为 `.xlsx`
- 仅包含失败记录，不含成功和跳过记录
- 字段：姓名、手机号、错误信息

## 9. 原型交互说明

由于当前为静态 HTML 原型，实际 Excel 解析由后端完成，前端做以下模拟：

- 点击"下载模板"下载标准 `.csv` 模板
- 选择任意 `.xls/.xlsx` 文件后"导入"按钮可用
- 导入结果按文件名关键字模拟：
  - 文件名含"失败/error/fail" → 模拟 3 条失败记录
  - 文件名含"跳过/skip/重复" → 模拟 2 条成功 + 2 条跳过
  - 其他文件名 → 模拟 3 条全部成功
- 成功数据以 mock 形式写入"已选考生"列表并刷新页面展示
- 失败时提供错误文件下载
