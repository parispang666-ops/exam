# 新增考试-集团 PRD

## 1. 文档范围

本文档仅描述"新增考试-集团"页面本次新增功能范围，包括集团跨公司创建考试、人员和部门范围选择、已选结果分组回显、免考人员配置、人员快照生成和新入职人员处理。

原有考试基本信息、考试设置、题目配置、步骤条流转、通用表单校验等基础能力沿用既有规则，本文不重复说明。

> 本页面PRD需配合[《考试功能-整体PRD》](./考试功能-整体PRD.md)阅读，全局业务规则、数据模型、角色权限定义和核心流程参见整体PRD。

## 2. 使用角色

- 集团管理员：在集团端创建考试，选择集团范围内的人员或部门作为考试对象，并配置免考范围。
- 系统：根据管理员选择的考试对象生成创建考试时人员快照，用于后续草稿发布前人员变更校验。

## 3. 功能清单

| 模块 | 功能 | 功能描述 |
|------|------|----------|
| 考生范围 | 集团跨公司创建 | 支持集团管理员在集团范围内跨公司选择考试对象。 |
| 考生范围 | 选择人员和部门 | 本次新增范围仅支持选择人员和部门范围。 |
| 已选结果 | 按公司分组回显 | 已选人员、部门按公司分组展示，并展示对应数量。 |
| 人员选择器 | 选择和查看已选 | 支持选人员、选部门、查看已选、全部移除，并将选择结果回写页面。 |
| 免考人员 | 免考范围配置 | 支持配置免考人员和免考部门范围，并回显已选结果。 |
| 人员快照 | 创建快照 | 创建考试草稿时生成创建考试时人员快照。 |
| 发布校验 | 快照基准 | 草稿发布前以创建或最近一次编辑后的人员快照为基准进行变更校验。 |

## 4. 需求说明

### 4.1 考试对象范围

| 规则项 | 页面要求 |
|--------|----------|
| 创建范围 | 支持集团范围内跨公司创建考试。 |
| 选择对象 | 本次新增功能仅支持选择人员和部门范围。 |
| 去重规则 | 系统自动对重复选择的人员进行去重，确保同一考试内同一人员只进入一次考生范围。 |
| 全部人员 | 选择"组织内全部人员"时，隐藏已选人员、已选部门和免考人员配置区。 |
| 部分人员 | 选择"仅部分人员"时，展示选择入口、已选人员、已选部门和免考人员配置区。 |

### 4.2 人员选择器

#### 4.2.1 打开入口与选择范围

| 项目 | 规则 |
|------|------|
| 考生范围选择入口 | 点击考生范围中的"选择"按钮，打开集团人员选择器，用于选择本次考试对象。选择按钮旁展示"共计人员：0人"，基于已选人员和部门解析后去重统计。 |
| 免考范围选择入口 | 点击免考人员区域的"选择"按钮，打开集团人员选择器，用于选择本次考试的免考对象。选择按钮旁展示"共计人员：0人"，基于已选免考人员和免考部门解析后去重统计。 |
| 查看已选入口 | 已选人员、已选部门、免考人员区域均支持进入查看已选，用于核对和维护已选结果。 |
| 选择范围 | 支持在集团范围内跨公司选择人员和部门。 |
| 选择对象 | 本次新增功能仅支持选择人员和部门范围。 |

#### 4.2.2 选择模式

| 模式 | 规则 |
|------|------|
| 选人员 | 支持按人员维度选择考试对象；可结合姓名、部门名称等条件进行查询，选择后同步进入右侧已选人员区域。 |
| 选部门 | 支持按部门维度选择考试对象；部门选择结果同步进入右侧已选部门区域。 |
| 部门级联 | 选择部门时支持按公司和部门层级展开，便于集团跨公司选择时定位部门。部门树按组织架构配置的"排序"字段升序排列，同一公司/父级下的部门按排序字段排列，组织架构调整排序后同步更新。 |
| 包含下级部门员工 | 选择部门范围时支持按业务需要包含下级部门员工，用于形成完整的部门考试对象范围。 |
| 已选数量 | 右侧已选区域分别展示人员数量和部门数量，选择、取消选择或移除后实时刷新。 |

#### 4.2.3 查询条件

| 所在区域 | 查询项 | 查询规则 |
|----------|--------|----------|
| 选人员列表 | 姓名、部门名称 | 姓名支持按输入内容查询；部门名称支持按公司和部门层级选择后查询。 |
| 选部门列表 | 部门名称 | 支持按部门名称查询部门树，匹配到的部门及其上级层级需保留展示。 |
| 查看已选人员 | 姓名、部门 | 仅在已选人员范围内查询，不影响未选人员列表。 |
| 查看已选部门 | 部门 | 仅在已选部门范围内查询，不影响未选部门列表。 |
| 查询按钮 | 执行当前条件查询 | 点击后刷新当前列表结果、已选状态和分页。 |
| 重置按钮 | 清空当前查询条件 | 点击后恢复当前列表默认结果，并保留已选结果。 |

#### 4.2.4 列表展示项

| 列表区域 | 展示项 | 展示规则 |
|----------|--------|----------|
| 选人员列表 | 勾选框、姓名、所属公司、所属部门 | 支持单个勾选、本页全选、分页展示；已选人员在列表中保持选中状态。 |
| 选部门列表 | 勾选框、部门名称、部门人数、部门层级 | 以部门层级方式展示；勾选部门时需联动右侧已选部门数量。 |
| 右侧已选人员 | 已选人员标签、人员数量 | 展示已选人员数量和人员标签，支持从标签上移除单个人员。 |
| 右侧已选部门 | 已选部门标签、部门数量 | 展示已选部门数量和部门标签，支持从标签上移除单个部门。 |
| 查看已选人员列表 | 勾选框、姓名、所属公司、所属部门、操作 | 仅展示已选人员；支持单个移除和批量删除。 |
| 查看已选部门列表 | 勾选框、部门名称、所属公司、人数、操作 | 仅展示已选部门；支持单个移除和批量删除。 |
| 空状态 | 暂无数据提示 | 当前查询或已选范围无数据时，列表展示空状态提示。 |

#### 4.2.5 查看已选与移除

| 操作 | 规则 |
|------|------|
| 查看已选人员 | 展示当前已选人员清单，支持继续核对和移除。 |
| 查看已选部门 | 展示当前已选部门清单，支持继续核对和移除。 |
| 单个移除 | 支持从已选区域移除单个人员或单个部门。 |
| 全部移除 | 支持清空当前范围内已选人员或部门，清空后同步刷新数量和页面回显。 |
| 确认返回 | 点击确定后，将当前已选人员、部门以及系统解析出的人员范围返回新增考试页面。 |
| 取消或关闭 | 取消或关闭人员选择器时，不更新新增考试页面当前已选结果。 |

#### 4.2.6 返回与回显规则

| 项目 | 规则 |
|------|------|
| 人员信息 | 返回人员需包含人员 ID、姓名、公司、部门、人员状态等信息，用于创建考试时人员快照和后续发布校验。 |
| 部门信息 | 返回部门需包含部门名称、所属公司、部门范围等信息，用于页面回显和人员范围解析。 |
| 去重规则 | 人员和部门范围解析后，如出现同一人员重复命中，系统需自动去重。 |
| 页面回显 | 选择结果返回页面后，立即刷新已选人员、已选部门、免考人员区域的数量和标签，同步更新考生范围和免考人员去重后共计人员数。 |
| 分组展示 | 集团端已选人员和部门需按公司分组展示，便于管理员核对跨公司选择结果。 |

### 4.3 已选人员和部门回显

| 展示对象 | 展示规则 |
|----------|----------|
| 已选人员 | 展示在"已选人员"区域，按公司分组展示人员标签和人数。 |
| 已选部门 | 展示在"已选部门"区域，按公司分组展示部门标签和部门数量。 |
| 单个移除 | 支持移除单个人员或单个部门。 |
| 全部移除 | 支持清空当前已选人员和部门范围。 |
| 可见性 | 未选择人员时，不展示已选人员区域；未选择部门时，不展示已选部门区域。 |
| 超量展示 | 页面已选人员、已选部门或免考已选结果超过 10 条时，页面最多展示前 10 条标签，并展示"查看更多 X 项"入口；X = 当前已选总条数 - 10。 |
| 查看更多 | 点击"查看更多 X 项"时，等同于点击当前区域的"查看已选"按钮，打开人员选择器的已选视图，用于查看和维护完整已选结果。 |

### 4.4 免考人员配置

| 项目 | 规则 |
|------|------|
| 配置入口 | 点击"免考人员"区域的"选择"按钮，打开人员选择器。 |
| 免考对象 | 支持选择免考人员和免考部门范围。 |
| 结果回显 | 页面展示已选免考人员数量、免考部门数量，免考人员选择按钮旁展示去重后共计人员数，并按公司分组展示。 |
| 查看已选 | 存在免考对象时，支持查看已选免考人员或免考部门。 |
| 超量展示 | 免考已选人员或免考已选部门超过 10 条时，页面最多展示前 10 条标签，并展示"查看更多 X 项"入口；X = 当前免考已选总条数 - 10。 |
| 全部移除 | 支持一键清空所有免考人员和免考部门。 |
| 业务口径 | 免考范围需随考试对象一并保存，用于后续应考人数和考试流程口径计算。 |

### 4.5 人员快照生成

| 场景 | 快照规则 |
|------|----------|
| 创建草稿 | 创建考试草稿时，系统基于当前考试对象生成创建考试时人员快照。 |
| 编辑保存 | 后续每次编辑保存后，需覆盖原人员快照。 |
| 快照内容 | 快照需保留人员 ID、姓名、公司、部门、人员状态等信息。 |
| 校验用途 | 快照用于草稿发布前比对当前考试对象和创建或编辑时的人员差异。 |

### 4.6 新入职人员处理

- 新入职人员不会自动进入已创建考试。
- 当考试对象为部门范围时，如果草稿发布前存在转入部门或启用人员，需在发布弹窗"新增"分类中提示。
- 管理员需要在发布弹窗中确认人员变更后，考试才继续发布。

## 5. 交互设计

### 5.1 选择考生范围

- 选择"仅部分人员"后，页面展示人员和部门选择能力；选择人员后才展示已选人员区域，选择部门后才展示已选部门区域。
- 选择"组织内全部人员"后，页面隐藏已选人员、已选部门和免考人员配置区。
- 从人员选择器确认返回后，页面即时刷新已选结果。

### 5.2 查看和移除已选

- 点击"查看已选"时，打开人员选择器的已选视图。
- 当页面已选标签超过 10 条时，仅展示前 10 条，并显示"查看更多 X 项"；点击后打开对应区域的查看已选视图，X = 当前已选总条数 - 10。
- 点击单个标签的移除按钮时，仅移除该人员或部门。
- 点击"全部移除"时，清空当前区域已选结果，并同步刷新数量。

### 5.3 保存与快照

- 保存草稿或完成本步骤保存时，系统生成或更新人员快照。
- 保存后进入后续考试设置流程。
- 后续发布校验以最新创建或编辑快照为基准。

## 6. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 仅部分人员但未选择考试对象 | 提示管理员至少选择一名已选人员或一个已选部门。 |
| 人员选择器取消或关闭 | 不更新当前页面已选结果。 |
| 当前区域无已选结果 | "查看已选"和"全部移除"不可用或弱化展示。 |

## 7. 验收标准

| 验收项 | 验收标准 |
|--------|----------|
| 跨公司选择 | 集团新增考试支持跨公司选择人员和部门。 |
| 选择范围 | 本次新增功能仅支持人员和部门范围。 |
| 已选回显 | 已选人员、部门必须按公司分组展示，并展示数量。 |
| 已选超量展示 | 页面已选结果超过 10 条时最多展示 10 条标签，并展示"查看更多 X 项"；点击后打开对应查看已选视图，X = 当前已选总条数 - 10。 |
| 免考配置 | 支持选择免考人员和免考部门，并按公司分组回显。 |
| 人员选择器 | 支持选人员、选部门、查看已选、单个移除、全部移除和确认返回。 |
| 部门选择 | 支持按公司和部门层级选择部门，并支持包含下级部门员工。 |
| 返回结果 | 人员选择器返回结果必须能够支撑页面回显、人员快照和发布前变更校验。 |
| 查看已选 | 已选人员、已选部门、免考人员均支持查看已选。 |
| 移除能力 | 支持单个移除和全部移除已选结果。 |
| 快照生成 | 创建考试草稿时必须生成创建考试时人员快照。 |
| 快照覆盖 | 每次编辑保存后必须覆盖原人员快照。 |
| 快照内容 | 快照必须包含人员状态，并保留公司、部门等发布校验所需信息。 |
| 新入职处理 | 新入职人员不会自动进入已创建考试，部门范围新增人员在发布校验中提示。 |
