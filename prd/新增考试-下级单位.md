# 新增考试-普通租户/下级单位 PRD

## 1. 文档范围

本文档描述"新增考试-普通租户"和"新增考试-下级单位"两个页面的统一功能。普通租户与下级单位在创建考试时遵循同一套业务逻辑：管理员仅能在本单位及下属部门范围内创建考试，不可向上级单位或平级单位发送。由于两者的本质逻辑一致，本文档合并说明。

**适用范围**：普通租户（非集团）和下级单位管理员的考试创建流程，包括考生范围选择、已选人员/部门回显（不分公司的平铺展示）、免考人员配置、人员快照生成和新入职人员处理。

**与集团版的核心区别**：不展示所属公司字段，已选人员和部门不按公司分组展示，选择范围限定在本单位及下属部门。

> 本页面PRD需配合[《考试功能-整体PRD》](./考试功能-整体PRD.md)阅读，全局业务规则、数据模型、角色权限定义和核心流程参见整体PRD。

## 2. 使用角色

- **下级单位管理员**：处于集团层级下的成员单位管理员，在本单位端创建考试，选择本单位及下属部门范围内的考试对象，并配置免考范围。不可向上级单位或平级单位发送考试。
- **普通租户管理员**：无上下级层级关系的独立组织管理员，在本租户范围内创建考试，选择本租户及下属部门范围内的考试对象，并配置免考范围。不可向其他单位发送考试。
- **系统**：根据管理员选择的考试对象生成创建考试时人员快照，用于后续草稿发布前人员变更校验。

## 3. 统一规则

普通租户和下级单位在新增考试时，共享以下核心逻辑：

| 规则项 | 说明 |
|--------|------|
| **创建范围** | 仅限本单位及下属部门，不可跨平级单位、不可向上级单位发送。 |
| **人员选择器** | 两类租户均使用同一人员选择器（`人员选择-下级单位.html`），选择范围自动限定为当前租户的组织节点及下属。 |
| **所属公司** | 不展示"所属公司"列，选择结果均不区分公司维度。 |
| **已选回显** | 已选人员、部门、免考结果均按平铺标签展示，不按公司分组。 |
| **页面差异** | 两个页面在交互、回显、快照生成等环节完全一致，仅页面标题（`新增考试-下级单位.html` / `新增考试-普通租户.html`）和嵌入上下文不同。 |

## 4. 功能清单

| 模块 | 功能 | 功能描述 |
|------|------|----------|
| 考生范围 | 本单位范围内创建 | 支持管理员在本单位及下属部门范围内创建考试，不可跨上级或平级单位。 |
| 考生范围 | 选择人员和部门 | 本次新增范围仅支持选择人员和部门范围。 |
| 已选结果 | 不分公司回显 | 已选人员按姓名标签展示、已选部门按名称标签展示，不按公司分组。 |
| 人员选择器 | 选择和查看已选 | 支持选人员、选部门、查看已选、全部移除，并将选择结果回写页面。 |
| 免考人员 | 免考范围配置 | 支持配置免考人员和免考部门范围，并回显已选结果。 |
| 人员快照 | 创建快照 | 创建考试草稿时生成创建考试时人员快照。 |
| 发布校验 | 快照基准 | 草稿发布前以创建或最近一次编辑后的人员快照为基准进行变更校验。 |

## 5. 需求说明

### 5.1 考试对象范围

| 规则项 | 页面要求 |
|--------|----------|
| 创建范围 | 仅支持本单位及下属部门范围内创建考试。下级单位不可向上级单位或平级单位发送考试；普通租户不可向其他单位发送考试。 |
| 选择对象 | 本次新增功能仅支持选择人员和部门范围。 |
| 去重规则 | 系统自动对重复选择的人员进行去重，确保同一考试内同一人员只进入一次考生范围。 |
| 全部人员 | 选择"组织内全部人员"时，隐藏已选人员、已选部门和免考人员配置区。 |
| 部分人员 | 选择"仅部分人员"时，展示选择入口、已选人员、已选部门和免考人员配置区。 |
| 所属公司 | 不展示所属公司字段，已选结果不按公司划分区域。 |

### 5.2 人员选择器

#### 5.2.1 打开入口与选择范围

| 项目 | 规则 |
|------|------|
| 考生范围选择入口 | 点击考生范围中的"选择"按钮，打开人员选择器，用于选择本次考试对象。 |
| 免考范围选择入口 | 点击免考人员区域的"选择"按钮，打开人员选择器，用于选择本次考试的免考对象。 |
| 查看已选入口 | 已选人员、已选部门均支持进入查看已选，用于核对和维护已选结果。 |
| 选择范围 | 仅支持在本单位及下属部门范围内选择人员和部门。下级单位不可向上级或平级单位选择，普通租户不可向其他单位选择。 |
| 选择对象 | 本次新增功能仅支持选择人员和部门范围。 |
| 人员选择器页面 | 两类租户均使用 `人员选择-下级单位.html`，不支持跨公司/跨单位选择。 |

#### 5.2.2 选择模式

| 模式 | 规则 |
|------|------|
| 选人员 | 支持按人员维度选择考试对象；可结合姓名、部门名称等条件进行查询，选择后同步进入右侧已选人员区域。 |
| 选部门 | 支持按部门维度选择考试对象；部门选择结果同步进入右侧已选部门区域。 |
| 部门级联 | 选择部门时支持按本单位部门层级展开，不涉及跨单位层级。部门树按组织架构配置的"排序"字段升序排列，同一父级下的部门按排序字段排列，组织架构调整排序后同步更新。 |
| 包含下级部门员工 | 选择部门范围时支持按业务需要包含下级部门员工，用于形成完整的部门考试对象范围。 |
| 已选数量 | 右侧已选区域分别展示人员数量和部门数量，选择、取消选择或移除后实时刷新。 |

#### 5.2.3 查询条件

| 所在区域 | 查询项 | 查询规则 |
|----------|--------|----------|
| 选人员列表 | 姓名、部门名称 | 姓名支持按输入内容查询；部门名称支持按本单位部门层级选择后查询。 |
| 选部门列表 | 部门名称 | 支持按部门名称查询部门树，匹配到的部门及其上级层级需保留展示。 |
| 查看已选人员 | 姓名、部门 | 仅在已选人员范围内查询，不影响未选人员列表。 |
| 查看已选部门 | 部门 | 仅在已选部门范围内查询，不影响未选部门列表。 |
| 查询按钮 | 执行当前条件查询 | 点击后刷新当前列表结果、已选状态和分页。 |
| 重置按钮 | 清空当前查询条件 | 点击后恢复当前列表默认结果，并保留已选结果。 |

#### 5.2.4 列表展示项

| 列表区域 | 展示项 | 展示规则 |
|----------|--------|----------|
| 选人员列表 | 勾选框、姓名、所属部门 | 支持单个勾选、本页全选、分页展示；已选人员在列表中保持选中状态。**不展示所属公司列。** |
| 选部门列表 | 勾选框、部门名称、部门人数、部门层级 | 以部门层级方式展示；勾选部门时需联动右侧已选部门数量。 |
| 右侧已选人员 | 已选人员标签、人员数量 | 展示已选人员数量和人员标签，支持从标签上移除单个人员。 |
| 右侧已选部门 | 已选部门标签、部门数量 | 展示已选部门数量和部门名称标签（仅展示部门名称，不含公司名），支持从标签上移除单个部门。 |
| 查看已选人员列表 | 勾选框、姓名、所属部门、操作 | 仅展示已选人员；支持单个移除和批量删除。不展示所属公司列。 |
| 查看已选部门列表 | 勾选框、部门名称、人数、操作 | 仅展示已选部门；支持单个移除和批量删除。不展示所属公司列。 |
| 空状态 | 暂无数据提示 | 当前查询或已选范围无数据时，列表展示空状态提示。 |

#### 5.2.5 查看已选与移除

| 操作 | 规则 |
|------|------|
| 查看已选人员 | 展示当前已选人员清单，支持继续核对和移除。 |
| 查看已选部门 | 展示当前已选部门清单，支持继续核对和移除。 |
| 单个移除 | 支持从已选区域移除单个人员或单个部门。 |
| 全部移除 | 支持清空当前范围内已选人员或部门，清空后同步刷新数量和页面回显。 |
| 确认返回 | 点击确定后，将当前已选人员、部门以及系统解析出的人员范围返回新增考试页面。 |
| 取消或关闭 | 取消或关闭人员选择器时，不更新新增考试页面当前已选结果。 |

#### 5.2.6 返回与回显规则

| 项目 | 规则 |
|------|------|
| 人员信息 | 返回人员需包含人员 ID、姓名、部门、人员状态等信息，用于创建考试时人员快照和后续发布校验。不返回所属公司。 |
| 部门信息 | 返回部门需包含部门名称、部门范围等信息，用于页面回显和人员范围解析。不返回所属公司。 |
| 去重规则 | 人员和部门范围解析后，如出现同一人员重复命中，系统需自动去重。 |
| 页面回显 | 选择结果返回页面后，立即刷新已选人员、已选部门的数量和标签。部门区域不显示时，隐藏整个已选部门区块。 |
| 不分组展示 | 已选人员和部门不按公司分组展示，以平铺标签形式呈现。 |

### 5.3 已选人员和部门回显

| 展示对象 | 展示规则 |
|----------|----------|
| 已选人员 | 展示在"已选人员"区域，以姓名标签平铺展示，不按公司分组。标签上仅显示姓名，hover 可查看完整部门路径。 |
| 已选部门 | 展示在"已选部门"区域，以部门名称标签平铺展示。 |
| 单个移除 | 支持移除单个人员或单个部门。 |
| 全部移除 | 支持清空当前已选人员和部门范围。 |
| 可见性 | 未选择人员时，不展示已选人员区域；未选择部门时，不展示已选部门区域。 |
| 数量展示 | 已选人员区域标题旁展示人员数量（如"1人"），部门区域标题旁展示部门数量（如"0个部门"）。 |
| 超量展示 | 页面已选人员、已选部门或免考已选结果超过 10 条时，页面最多展示前 10 条标签，并展示"查看更多 X 项"入口；X = 当前已选总条数 - 10。 |
| 查看更多 | 点击"查看更多 X 项"时，等同于点击当前区域的"查看已选"按钮，打开人员选择器的已选视图，用于查看和维护完整已选结果。 |

### 5.4 免考人员配置

| 项目 | 规则 |
|------|------|
| 配置入口 | 点击"免考人员"区域的"选择"按钮，打开人员选择器。 |
| 免考对象 | 支持选择免考人员和免考部门范围。 |
| 结果回显 | 页面展示已选免考人员数量、免考部门数量。已选结果不按公司分组，以标签形式平铺展示。 |
| 查看已选 | 存在免考对象时，支持查看已选免考人员或免考部门。 |
| 超量展示 | 免考已选人员或免考已选部门超过 10 条时，页面最多展示前 10 条标签，并展示"查看更多 X 项"入口；X = 当前免考已选总条数 - 10。 |
| 全部移除 | 支持一键清空所有免考人员和免考部门。 |
| 业务口径 | 免考范围需随考试对象一并保存，用于后续应考人数和考试流程口径计算。 |

### 5.5 人员快照生成

| 场景 | 快照规则 |
|------|----------|
| 创建草稿 | 创建考试草稿时，系统基于当前考试对象生成创建考试时人员快照。 |
| 编辑保存 | 后续每次编辑保存后，需覆盖原人员快照。 |
| 快照内容 | 快照需保留人员 ID、姓名、部门、人员状态等信息。不含所属公司。 |
| 校验用途 | 快照用于草稿发布前比对当前考试对象和创建或编辑时的人员差异。 |

### 5.6 新入职人员处理

- 新入职人员不会自动进入已创建考试。
- 当考试对象为部门范围时，如果草稿发布前存在转入部门或启用人员，需在发布弹窗"新增"分类中提示。
- 管理员需要在发布弹窗中确认人员变更后，考试才继续发布。

## 6. 交互设计

### 6.1 选择考生范围

- 选择"仅部分人员"后，页面展示人员选择按钮、已选人员区域（有人员时）和已选部门区域（有部门时）。
- 选择"组织内全部人员"后，页面隐藏选择按钮、已选人员区域、已选部门区域和免考人员配置区。
- 从人员选择器确认返回后，页面即时刷新已选结果。

### 6.2 查看和移除已选

- 点击"查看已选"时，打开人员选择器的已选视图。
- 当页面已选标签超过 10 条时，仅展示前 10 条，并显示"查看更多 X 项"；点击后打开对应区域的查看已选视图，X = 当前已选总条数 - 10。
- 点击单个标签的移除按钮时，仅移除该人员或部门，其他标签保持不动。
- 点击"全部移除"时，清空当前区域已选结果，并同步刷新数量。

### 6.3 保存与快照

- 保存草稿或完成本步骤保存时，系统生成或更新人员快照。
- 保存后进入后续考试设置流程。
- 后续发布校验以最新创建或编辑快照为基准。

### 6.4 页面跳转

- 点击"上一步"返回基本信息步骤。
- 点击"保存并下一步"进入考试设置步骤，保存前需校验至少选择了一名已选人员或一个已选部门。

## 7. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 仅部分人员但未选择考试对象 | 提示管理员至少选择一名已选人员或一个已选部门。 |
| 人员选择器取消或关闭 | 不更新当前页面已选结果。 |
| 当前区域无已选结果 | "查看已选"和"全部移除"按钮置灰不可用。 |
| 已选人员区域无人员时 | 隐藏整个已选人员区域。 |
| 已选部门区域无部门时 | 隐藏整个已选部门区域。 |
