# 考试列表-下级单位 PRD（产品需求文档）

## 1. 文档范围

本文档仅描述"考试列表-下级单位"相对于通用考试列表新增的功能范围，包括混合考试来源展示、上级下发考试标签、操作权限分层和分类树上级下发节点。

原有列表查询、分页、新增、编辑、复制、删除、批量删除等基础能力沿用既有规则，本文不重复说明。

> 本页面PRD需配合[《考试功能-整体PRD》](./考试功能-整体PRD.md)阅读，全局业务规则、数据模型、角色权限定义和核心流程参见整体PRD。

## 2. 使用角色

- 下级单位管理员：在本单位端查看本单位自建考试和上级下发考试，对本单位自建考试执行完整管理操作，对上级下发考试仅执行管理操作。
- 系统：区分考试来源（本单位/上级下发），并按来源控制操作权限。

## 3. 功能清单

| 模块 | 功能 | 功能描述 |
|------|------|----------|
| 考试列表 | 混合考试来源 | 列表同时展示本单位自建考试和上级下发考试。 |
| 考试列表 | 上级下发标签 | 上级下发考试行首展示「上级下发」标签，视觉区分考试来源。 |
| 行内操作 | 操作权限分层 | 上级下发考试仅支持「统计」，不支持预览、发布、编辑、复制、删除、批量删除；本单位自建考试支持完整操作。 |
| 考试分类树 | 上级下发节点 | 左侧分类树独立展示「上级下发」节点，支持按来源筛选。 |
| 行内操作 | 查看集团考试详情 | 上级下发考试点击名称，可查看集团版考试详情。 |

## 4. 需求说明

### 4.1 混合考试来源

- 页面数据范围同时包含本单位自建考试和上级下发考试。
- 上级下发考试由集团或上级组织下发，在下级单位列表中展示。
- 本单位自建考试由本级管理员创建，支持下级单位完整管理操作。

### 4.2 上级下发标签

- 上级下发考试在考试名称前展示「上级下发」标签。
- 标签样式以蓝色背景边框区分，视觉上与本级自建考试明确分离。

### 4.3 操作权限分层

#### 4.3.1 上级下发考试操作权限

| 操作 | 支持状态 | 说明 |
|------|----------|------|
| 统计 | ✅ 支持 | 跳转至下级单位统计页查看考试数据。 |
| 查看详情（名称链接） | ✅ 支持 | 点击考试名称可查看集团版考试详情。 |
| 预览 | ❌ 不支持 | 不展示预览按钮。 |
| 发布 | ❌ 不支持 | 不展示发布入口。 |
| 编辑 | ❌ 不支持 | 不展示编辑入口。 |
| 复制 | ❌ 不支持 | 不展示复制入口。 |
| 删除 | ❌ 不支持 | 不展示删除入口。 |
| 批量删除 | ❌ 不支持 | 复选框置灰，不可勾选。 |

#### 4.3.2 本单位自建考试操作权限

| 操作 | 支持状态 | 说明 |
|------|----------|------|
| 预览 | ✅ 支持 | 预览考试详情。 |
| 统计 | ✅ 支持 | 查看考试统计数据。 |
| 发布 | ✅ 支持 | 草稿考试支持发布。 |
| 复制 | ✅ 支持 | 复制考试。 |
| 编辑 | ✅ 支持 | 编辑考试。 |
| 删除 | ✅ 支持 | 删除考试。 |
| 批量删除 | ✅ 支持 | 勾选后批量删除。 |

### 4.4 分类树上级下发节点

- 左侧考试分类树顶部独立展示「上级下发」节点。
- 点击「上级下发」节点，列表仅筛选展示上级下发考试。
- 「上级下发」节点与「全部分类」及具体分类节点并列，支持独立选择。

### 4.5 草稿发布规则（本单位自建考试）

- 仅本单位自建草稿考试支持发布。
- 发布前需校验人员变更，规则与集团版一致。
- 有人员变更时展示发布弹窗，管理员确认后发布。
- 上级下发考试不涉及发布操作。

## 5. 交互设计

### 5.1 列表展示

- 上级下发考试与本单位自建考试在同一列表混合展示。
- 上级下发考试行首展示「上级下发」蓝色标签。
- 上级下发考试名称可点击，查看集团版考试详情。
- 上级下发考试复选框置灰不可选。

### 5.2 操作按钮

- 上级下发考试操作列仅展示「统计」按钮。
- 本单位自建考试操作列展示「预览」「统计」及更多操作。
- 本单位草稿考试额外展示「发布」入口。

### 5.3 分类树交互

- 默认选中「全部分类」，展示所有考试。
- 点击「上级下发」节点，筛选上级下发考试。
- 点击具体分类节点，筛选该分类下的全部考试（含上级下发和本单位自建）。

## 6. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 上级下发考试操作受限 | 对不支持的操作，隐藏对应按钮，复选框置灰。 |
| 上级下发考试批量删除 | 复选框不可选，勾选时不纳入批量删除范围。 |

## 7. 验收标准

| 验收项 | 验收标准 |
|--------|----------|
| 混合展示 | 页面同时展示本单位自建考试和上级下发考试。 |
| 上级下发标签 | 上级下发考试行首展示「上级下发」标签。 |
| 名称链接 | 上级下发考试名称可点击，查看集团版考试详情。 |
| 操作权限 | 上级下发考试仅展示「统计」按钮；不展示预览、发布、编辑、复制、删除入口。 |
| 复选框 | 上级下发考试复选框置灰不可选，批量删除不纳入。 |
| 分类树节点 | 分类树顶部独立展示「上级下发」节点，点击筛选上级下发考试。 |
| 本单位考试 | 本单位自建考试支持预览、统计、发布、复制、编辑、删除、批量删除等完整操作。 |
